巴哈推文實況
=============

[![.github/workflows/build.yml](https://github.com/eight04/baha-live-push/actions/workflows/build.yml/badge.svg)](https://github.com/eight04/baha-live-push/actions/workflows/build.yml)

一個用於巴哈姆特電玩資訊站的推文實況腳本，自動讀取新推文並加入頁面中

安裝後會在回覆欄加入一行狀態與按鈕

Install
-------

[Greasyfork](https://greasyfork.org/zh-TW/scripts/580574-%E5%B7%B4%E5%93%88%E6%8E%A8%E6%96%87%E5%AF%A6%E6%B3%81)

Compatibility
-------------

* CSS Nesting - Firefox 117
* aspect-ratio - Firefox 89
* object-fit - Firefox 36
* :has - Firefox 121
* ?. - Firefox 74
